## データ構造

### タスクシート

| カラム | 型 | 説明 |
|--------|------|------|
| タスクID | string (UUID先頭8文字) | 一意識別子 |
| タスク名 | string | タスクの名称 |
| 締め切り | date (yyyy-MM-dd) | 期限日 |
| ペナルティID | string | 紐付けペナルティの識別子（任意） |
| 完了 | boolean / "完了" | 完了状態 |
| 作成日 | datetime | 作成日時 |
| 完了日 | datetime | 完了日時（完了時に記録） |

### ペナルティシート

| カラム | 型 | 説明 |
|--------|------|------|
| ペナルティID | string (UUID先頭8文字) | 一意識別子 |
| ペナルティ名 | string | ペナルティの名称 |
| 詳細 | string | ペナルティの詳しい内容（任意） |

### ペナルティ履歴シート

| カラム | 型 | 説明 |
|--------|------|------|
| 履歴ID | string (UUID先頭8文字) | 一意識別子 |
| タスクID | string | 対象タスクの識別子 |
| タスク名 | string | 対象タスクの名称（スナップショット） |
| ペナルティID | string | 対象ペナルティの識別子 |
| ペナルティ名 | string | 対象ペナルティの名称（スナップショット） |
| 発動日 | datetime | ペナルティ発動日時 |
| 実行済み | boolean / "実行済み" | ペナルティ実行状態 |

## API エンドポイント一覧

すべて GAS Web API 経由。`action` クエリパラメータでルーティング。

| action | メソッド | パラメータ | 説明 |
|--------|---------|-----------|------|
| `getAll` | GET | なし | タスク・ペナルティ・履歴を一括取得（初期読み込み用） |
| `getTasks` | GET | なし | タスク一覧取得 |
| `addTask` | GET | name, deadline, penaltyId | タスク追加 |
| `completeTask` | GET | taskId | タスクを完了にする |
| `deleteTask` | GET | taskId | タスクを削除 |
| `getPenalties` | GET | なし | ペナルティ一覧取得 |
| `addPenalty` | GET | name, detail | ペナルティ追加 |
| `updatePenalty` | GET | penaltyId, name, detail | ペナルティ編集 |
| `deletePenalty` | GET | penaltyId | ペナルティ削除 |
| `getHistory` | GET | なし | ペナルティ履歴一覧取得 |
| `triggerPenalty` | GET | taskId | 期限超過タスクのペナルティを発動 |
| `markHistoryDone` | GET | historyId | ペナルティ履歴を「実行済み」にする |

## 画面構成

### 1. 設定パネル

- GAS デプロイ URL を入力・保存
- `localStorage` に永続化
- 未設定時に自動表示

### 2. ダッシュボード

- **月間カレンダー**（最上部に配置）:
  - 前月/翌月ナビゲーション、「今日」ボタン
  - 締切日のあるセルにタスク名を表示（最大3件、超過分は「+N件」表示）
  - 色分け: 期限切れ=赤、当日/3日以内=黄、通常=紫、完了=緑取消線
  - 曜日ヘッダー（日=赤、土=青）、今日のセルは紫枠
- **統計カード**: 未完了タスク数、期限切れ数、未実行ペナルティ数、完了済み数
- **期限切れアラート**: 期限超過タスクを赤く表示（曜日付き）。完了ボタン・ペナルティ発動ボタン付き
- **期限が近いタスク**: 未完了かつ期限内のタスクを期限順に最大5件表示

### 3. タスク一覧

- 未完了タスクを期限順に表示（期限切れ → 期限近い → 期限なし）
- 完了済みタスクは下部に折りたたみ表示
- 各タスクに: 完了ボタン、削除ボタン、期限超過時はペナルティ発動ボタン
- 「+ 追加」ボタンでモーダル表示（タスク名・締め切り・ペナルティ選択）

### 4. ペナルティ一覧

- 登録済みペナルティのカード一覧
- 各ペナルティに: 編集ボタン、削除ボタン
- 「+ 追加」ボタンでモーダル表示（ペナルティ名・詳細）

### 5. ペナルティ履歴

- 発動済みペナルティを新しい順に表示
- 未実行: 「実行済みにする」ボタン表示
- 実行済み: 「実行済み」タグ表示
- ナビゲーションに未実行件数のバッジ表示

## UI/UX 仕様

- **カラースキーム**: 紫系プライマリ（#6c5ce7）、赤系デンジャー、緑系サクセス
- **カードデザイン**: 左ボーダー色で状態を示す（通常:紫、期限切れ:赤、完了:緑）
- **モーダル**: オーバーレイ背景クリックで閉じる
- **レスポンシブ**: 480px以下でフォームを縦並び、統計カードを2列に
- **タブ切り替え**: 下線アクティブ表示のナビゲーション
- **締め切り表示**: すべての締切表示に曜日を含む。3日以内は「あとN日（曜日）」、当日は「今日（曜日）まで！」、超過は「期限切れ（N日超過）日付（曜日）」、通常は「日付（曜日）」
