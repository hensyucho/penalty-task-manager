# 自堕落な人向けタスク管理 — テスト項目表

> **機能追加時はテスト項目も追記すること。**

---

## 1. 初期セットアップ

- [ ] GAS で `setupSheets` を実行すると「タスク」「ペナルティ」「ペナルティ履歴」シートが作成される
- [ ] 既にシートが存在する場合、重複作成されない
- [ ] デフォルトの「Sheet1」/「シート1」が削除される

## 2. API 接続

- [ ] 正しいデプロイURLを入力して「保存して接続」→「接続済み」と表示される
- [ ] 不正なURLを入力 → エラーメッセージが表示される
- [ ] URLが `localStorage` に保存され、リロード後も再接続される
- [ ] 設定ボタンからURLを変更できる

## 3. タスク管理

### 追加
- [ ] タスク名・締め切りを入力して追加 → タスク一覧に表示される
- [ ] ペナルティを紐付けて追加 → タスクにペナルティタグが表示される
- [ ] タスク名未入力 → バリデーションエラー
- [ ] 締め切り未入力 → バリデーションエラー
- [ ] 締め切りのデフォルトが翌日になっている

### 完了
- [ ] 完了ボタン → 確認ダイアログ → 完了済みに変わる
- [ ] 完了済みタスクは下部に薄く表示される
- [ ] ダッシュボードの統計が更新される

### 削除
- [ ] 削除ボタン → 確認ダイアログ → タスクが消える
- [ ] 削除後、ダッシュボードの統計が更新される

### ペナルティ変更
- [ ] ペナルティ未設定タスクに「ペナルティ設定」ボタンが表示される
- [ ] ペナルティ設定済みタスクに「ペナルティ変更」ボタンが表示される
- [ ] モーダルで現在のペナルティが選択状態になっている
- [ ] ペナルティを選択して保存 → タスクにペナルティタグが表示される
- [ ] 「ペナルティなし」に変更 → ペナルティタグが消える
- [ ] 完了済みタスクにはペナルティ変更ボタンが表示されない

### 表示順序
- [ ] 未完了タスクは期限順（期限切れ → 期限近い → 期限なし）
- [ ] 完了済みタスクは未完了の下に表示

## 4. ペナルティ管理

### 追加
- [ ] ペナルティ名を入力して追加 → 一覧に表示される
- [ ] 詳細も入力可能（任意）
- [ ] ペナルティ名未入力 → バリデーションエラー
- [ ] 追加後、タスク追加モーダルのペナルティ選択肢に反映される

### 編集
- [ ] 編集ボタン → モーダルに既存値が入る → 保存で更新される

### 削除
- [ ] 削除ボタン → 確認ダイアログ → ペナルティが消える

## 5. ペナルティ発動・履歴

### 発動
- [ ] 期限切れタスクの「ペナルティ発動」ボタン → 確認ダイアログ → 履歴に記録される
- [ ] ペナルティ未設定のタスクには発動ボタンが表示されない
- [ ] ダッシュボードの期限切れアラートからも発動可能

### 履歴
- [ ] 発動済みペナルティが新しい順に表示される
- [ ] 「実行済みにする」ボタンで実行済みに変わる
- [ ] 未実行件数がナビゲーションのバッジに表示される
- [ ] 全て実行済みだとバッジが非表示になる

## 6. ダッシュボード

- [ ] 統計カード4種が正しい数値を表示する
- [ ] 期限切れタスクが赤いアラートで表示される
- [ ] 期限が近いタスク（未完了・期限内）が最大5件表示される
- [ ] タスクがない場合、お祝いメッセージが表示される

## 7. UI/UX

- [ ] モーダル外クリックで閉じる
- [ ] タブ切り替えが正常に動作する
- [ ] スマートフォン幅（480px以下）でレイアウトが崩れない
- [ ] 期限3日以内は「あとN日（曜日）」表示
- [ ] 期限当日は「今日（曜日）まで！」表示
- [ ] 期限超過は「期限切れ（N日超過）日付（曜日）」表示
- [ ] 通常の締切は「日付（曜日）」表示

## 8. カレンダー

- [ ] ダッシュボード最上部に月間カレンダーが表示される
- [ ] 前月・翌月ボタンで月を切り替えられる
- [ ] 「今日」ボタンで当月に戻れる
- [ ] 今日のセルが紫枠でハイライトされる
- [ ] 曜日ヘッダーが表示される（日=赤、土=青）
- [ ] 締切日のあるセルにタスク名が表示される（最大3件）
- [ ] 4件以上のタスクがある日は「+N件」と表示される
- [ ] タスクの色分け: 期限切れ=赤、3日以内/当日=黄、通常=紫、完了=緑取消線
- [ ] 前月・翌月の日付がグレーアウト表示される
- [ ] スマートフォン幅でもカレンダーが崩れない

## 9. エラーハンドリング

- [ ] API通信エラー時にアラートが表示される
- [ ] 存在しないタスクID/ペナルティIDでの操作がエラーを返す
- [ ] ローディング中はスピナーが表示される
