# 自堕落な人向けタスク管理 - セットアップ手順

## 概要

Googleスプレッドシートをデータベースとして使う、プライベートなタスク管理アプリです。
ブラウザで `index.html` を開くだけで使えます。

---

## 手順1: Googleスプレッドシートを作成

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 「空白のスプレッドシート」を新規作成
3. スプレッドシートの名前を「自堕落な人向けタスク管理」などに変更
4. 共有相手と共有する場合は、右上の「共有」ボタンから共有相手のGoogleアカウントを追加

---

## 手順2: Google Apps Script をセットアップ

1. スプレッドシートのメニューから **「拡張機能」→「Apps Script」** を開く
2. 左側の `コード.gs` の内容をすべて削除
3. `Code.gs` ファイルの内容をすべてコピーして貼り付ける
4. **保存**（Ctrl+S）

### 初期セットアップの実行

5. Apps Script エディタの上部で、実行する関数を `setupSheets` に変更
6. **「実行」** ボタンをクリック
7. 初回は「承認が必要です」と表示されるので：
   - 「権限を確認」をクリック
   - 自分のGoogleアカウントを選択
   - 「詳細」→「（安全でないページ）に移動」をクリック
   - 「許可」をクリック
8. スプレッドシートに「タスク」「ペナルティ」「ペナルティ履歴」の3つのシートが作成されればOK

---

## 手順3: Web API としてデプロイ

1. Apps Script エディタの右上 **「デプロイ」→「新しいデプロイ」** をクリック
2. 左の歯車アイコンから **「ウェブアプリ」** を選択
3. 以下の設定にする：
   - **説明**: 自堕落な人向けタスク管理API（任意）
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: **全員**（※URLを知っている人だけがアクセスできます）
4. **「デプロイ」** をクリック
5. 表示される **ウェブアプリのURL** をコピー（`https://script.google.com/macros/s/xxxxx/exec` の形式）

> **重要**: このURLは他人に教えないでください。URLを知っている人は誰でもデータにアクセスできます。

---

## 手順4: HTML アプリを設定

1. `index.html` をブラウザで開く（ダブルクリックまたはブラウザにドラッグ＆ドロップ）
2. 設定パネルが表示されるので、手順3でコピーしたURLを貼り付け
3. 「保存して接続」をクリック
4. 「接続済み」と表示されれば完了！

---

## 使い方

### タスクの追加
1. 「タスク」タブを開く
2. 「+ 追加」ボタンをクリック
3. タスク名・締め切り・ペナルティ（任意）を設定して追加

### ペナルティの登録
1. 「ペナルティ一覧」タブを開く
2. 「+ 追加」ボタンからペナルティを登録
3. タスク追加時に、登録したペナルティを選択できるようになる

### 期限切れタスクのペナルティ発動
1. ダッシュボードに期限切れタスクが赤く表示される
2. 「ペナルティを発動」ボタンでペナルティ履歴に記録される
3. ペナルティを実行したら「実行済みにする」ボタンで記録

### 共有相手との共有
- 同じ `index.html` ファイルを共有相手に渡す（またはOneDrive/Googleドライブで共有）
- 共有相手も同じデプロイURLを設定すれば、同じデータを見られる

---

## トラブルシューティング

### 「接続に失敗しました」と表示される
- デプロイURLが正しいか確認
- Apps Scriptのデプロイで「アクセスできるユーザー」が「全員」になっているか確認

### データが反映されない
- ブラウザをリロード（F5）してください
- 同時編集には対応していないため、片方が操作中はもう片方はリロードで最新を取得

### コードを修正した場合
- Apps Scriptで修正後、**「デプロイ」→「デプロイを管理」→ 鉛筆アイコン → バージョンを「新しいバージョン」に → 「デプロイ」** で更新
- URLは変わりません
